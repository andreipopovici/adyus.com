Intralinks R&D

Status: mostly covered under NDA, internal projects

Designed web and mobile prototypes to explore future directions for the company’s product offerings. I filled in as product manager for the Android and iOS mobile apps briefly.

ezsend.it

Status: almost ready, any day now :)

A long-running personal project. Since the UX for buying a stamp or shipping label online is not exactly optimized for single, quick use, I figured I’d do it.

This served as a learning project both for trying new backend technologies such as Meter.js and AWS Lambda, as well as optimizing the UI (several times) and dabbling with Material Design.

While a fully-functioning beta version was almost ready for launch, I decided it could still be improved further, thus it’s currently being redesigned from scratch.
